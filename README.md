# Información sobre la app

La temática de la página es la creación de un servicio de venta de pizzas, en donde el usuario interactúa con la aplicación mediante 2 formas, una es eligiendo la pizza y la cantidad que quiere comprar, se le proporcionará el precio, una imágen a modo de ilustración y el tipo de pizza, teniendo 4 opciones. Además el usuario dispone de un carrito que funciona como un componente hermano del componente donde se muestran las pizzas.
Al agregar el producto al carrito, en este se puede eliminar 1 pizza o se pueden eliminar todas las pizzas y formar un pedido de nuevo.
La idea es que no funcione el botón de confirmar compra si no hay ninguna pizza agregada al carrito.
Cuando se agregan pizzas y se confirma la compra, el usuario es redirigido a la página de contacto, donde proporcionará los datos para que se le haga el envío de las pizzas. Formulario creado en modo ReactiveForm. 
La aplicación dispone de todos los componentes necesarios e imprescindibles para su funcionamiento. Además se generaron servicios para poder enlazar un servicio de Api Rest y que quede una página completamente funcional.
